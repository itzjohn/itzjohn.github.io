Install / Setup Hexo

Created2020-11-30|Updated2020-11-30
|Post View:

About this post:

This is post will teach you how to install and setup Hexo on a Ubuntu OS device.

What is Hexo?

“Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other markup languages) and Hexo generates static files with a beautiful theme in seconds.” This blog uses Hexo’s blog framework with the help from NodeJS to generate everything, requires very little amount of programming experenice to set it up.

Requirements to setup our blog

  • Download & Install NodeJS - Executes JavaScript code outside a web browser, its used to generate the blog.
  • Download & Install Git - Version-control system for tracking changes in any set of files and to upload your files to github, more on this later
  • Download & Install Hexo - The blog’s framework

Download & Install NodeJS

  1. Open up your terminal
  2. Type in the following codes 
    curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash - sudo apt-get install -y nodejs
  3. Verify that NodeJS was installed correct by typing in node -v it should display something along the lines of v14.##.##

Download & Install Git

  1. Open up your terminal
  2. Type in the following codes 
    sudo apt-get install git
  3. Verify that Git was installed correct by typing in git --version it should display something along the lines of git --version 2.##.#
  4. To config it with some basic information, type in the following codes
    git config --global user.name "Enter Name Here"
git config --global user.email "Here Email Address Here"

Download & Install Hexo

*NodeJS needs to be installed first because it uses NodeJS.

  1. Open up your terminal
  2. Type in the following codes 
    npm install -g hexo-cli

Create your first blog

  1. Open up your terminal

  2. Type in the following codes 

    hexo init [folder name]
cd [folder name]
npm install

  3. Verify that everything was created correctly by typing ls in your terminal

    or going directly to the folder located in /home/[username]/[folder name]

  4. Lets create our first post by typing in hexo new post "post name" and it will notify you that a new file was created.

  5. Locate the file that was just created and open it up with any text editor. Input some random string of text under the --- and save the file.

  6. Head back to the terminal and type hexo sever

  7. Select a web browser and go to localhost:4000 to view your blog.

Final notes:

  • To stylize your post with images, links, lists, etc you can use markdown lang.
  • Make sure you read up on Hexo Documentation to better understand Hexo’s framework
  • You can uses other themes if the current one doesn’t suit your taste. I’m currently using Butterfly theme with some minor adjustments.
